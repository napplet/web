# Plan 145-01: Repair Production Type Boundaries

## Objective

Close scanner-reported type-safety warnings by replacing broad production assertions with local structural types and discriminated message handling.

## Tasks

<task id="145-01-01">
<title>Repair global mount assertions</title>

<action>
Replace `window as any` and `window as unknown as ...` mount casts with explicit structural window types for `window.napplet` and `window.nostrdb`.
</action>

<acceptance_criteria>

</acceptance_criteria>
</task>

</action>

<acceptance_criteria>

</acceptance_criteria>
