### Note on the local browser binary

`^1.59.1` resolved to playwright `1.61.0`, which expects Chromium revision 1228.
The shared `~/.cache/ms-playwright` cache only had revisions 1208/1217, so the
verification run used `executablePath: '/usr/bin/chromium'` (system Chromium per
CLAUDE.md) with crawler logic identical to the committed script. The committed
`scripts/check-links.mjs` is unmodified and uses the default browser resolution;
CI runs `playwright install --with-deps chromium` to fetch the matching 1228
build, so no `executablePath` override is needed there. The temporary
verification shim was created and removed inside `scripts/` (so the bare
`playwright` specifier resolved) and is not part of the commit.
